Specs of Solution


  1. The core of the solution is a NoSQL document database - either MongodB or Couchbase.
  2. In MongodB parlance, a record is called a "document".
  3. A NoSQL database offers the flexibility to include an arbitrary number of search tags with each document.
  4. Each document is a fully hierarchised JSON record.
  5. The full hierachisation makes possible context-sensitive searches.
