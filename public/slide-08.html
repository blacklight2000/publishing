Specs of Proposed Classification System


  1. Must include all existing media formats: hard copy, digital copy, audio, video, hypertext, etc.
  2. Must include all printed materials: pamphlets, tracts, newspapers, magazines, etc.
  3. Must be an open system, making provision for media formats that don't exist as yet.
  4. Must include complete spec of the material's physical location - eventually, for internal use only.
  5. Must be able to include an arbitrary number of search tags.
